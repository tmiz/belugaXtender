belugaXtender


Safari Extension for Beluga Group Chat on web access.
  • Find Beluga on tags
  • Post without clicking [Send] button( [Enter] key or [Shift]+[Enter] keys)
  • Counter Badge of Unread Messages
  • Open contained URL on background tab automatically


This extension has been inspired from Chrome Extension Beluga Helper (drikin's).

Download

You can download exntension binary(0.2.2) on this project.

Install

(1) Do double click BelugaXtender.safariextz. And permit to use extension on Safari.
(2) If you want to use with Growl, BelugaXtender requires GrowlSafariBridge.
do download from following website https://github.com/uasi/growl-safari-bridge/downloads and move GrowlSafariBridge.webplugin to ~/Library/Internet Plugins/
